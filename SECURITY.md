# Security Policy

## Supported Versions

Only the latest release of `openwiki` is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 0.0.1| :white_check_mark: |
| < 0.0.1 | :x:                |

## Reporting a Vulnerability

We take the security of `openwiki` seriously. If you find a security vulnerability, please do not disclose it publicly. Instead, please follow this process:

1. **Submit a Security Advisory:** Navigate to the **Security** tab of this repository on GitHub and submit a security advisory.
2. **Notify via Email:** Send an email to **security@langchain.dev** to notify the team that you have filed a security issue, including the repository name (`openwiki`).

We will acknowledge your report promptly and coordinate the publication of a fix and a security advisory.
